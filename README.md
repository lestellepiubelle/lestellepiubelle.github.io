# Le stelle più affascinanti del cielo

Un progetto di divulgazione scientifica dedicato ad articoli monografici su stelle particolarmente interessanti per storia evolutiva, massa, luminosità, caratteristiche fisiche insolite, possibilità di ospitare esopianeti abitabili ecc.

Al catalogo delle stelle è associato un dizionario pratico di astronomia e astrofisica, in continua evoluzione, nel quale le formule matematiche sono accompagnate da esempi applicativi che guidano il lettore verso una comprensione più immediata dei concetti.

Il sito è realizzato con Hugo e pubblicato tramite GitHub Pages.

## Struttura del progetto

- `content/` contiene gli articoli e le voci del dizionario in formato Markdown.
- `static/` contiene immagini e risorse statiche.
- `.github/workflows/` contiene la configurazione per la pubblicazione automatica.

## Tecnologie utilizzate

- Hugo
- Markdown
- GitHub Pages
- GitHub Actions
